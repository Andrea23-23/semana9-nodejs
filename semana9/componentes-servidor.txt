Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del sistema y permite ejecutar aplicaciones.

Node.js:
Es un entorno de ejecución para JavaScript. Permite crear y ejecutar aplicaciones del lado del servidor.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta práctica se utilizó el puerto 3000.

Logs:
Son registros de eventos y actividades del servidor. Ayudan a monitorear el funcionamiento y detectar errores.
